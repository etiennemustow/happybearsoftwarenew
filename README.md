# Happy Bear Software Sample Test

## Start-up Instructions

* Git clone this repo
* Change into the 'guestbook' directory
```
cd guestbook
```
* Start up the server
```
rails s -p $PORT -b $IP
```
* Open up your browser and enter `http://localhost:8080/` in the URL

## Running Tests

* Enter the following to run spec tests
```
bundle exec rspec
```
